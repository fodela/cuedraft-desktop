# Support

## Getting Help

- Usage questions and troubleshooting: open a [GitHub Issue](https://github.com/fodela/cuedraft-desktop/issues)
- Bugs: use the Bug Report template
- Feature requests: use the Feature Request template

## Before Opening an Issue

- Confirm you are on the latest commit or release
- Search existing issues for duplicates
- Include platform details (Windows/Linux, distro, desktop environment)
